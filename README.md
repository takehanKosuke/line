[追加機能]  
・グループ機能の追加  
・不適切ワードの設定を追加  
  ```
  bundle exec rails runner Task.count_of_inappropriate_words
  ```
  上記コマンドを実行することで不適切ワードの発言回数をカウント  
  １０回以上不適切ワードを発言した場合アカウントをバンするように設定
